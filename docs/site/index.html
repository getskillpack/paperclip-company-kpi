Company-wide KPI and spend, inside Paperclip.

Roll up cost_event streams, add a manual ledger, and track executive targets — one dashboard for operators and leadership.

Problem: spend and KPIs are scattered across logs and spreadsheets. Solution: trusted plugin state in Paperclip — monthly rollups, auditable ledger, C-level targets. MIT licensed.

  • Monthly cost rollups from cost_event.created
  • Manual income / expense lines
  • Executive KPI registry (see repo docs)

Install:

npm install @getskillpack/paperclip-company-kpi

Until the package is on the public registry, install from git: npm install github:getskillpack/paperclip-company-kpi#v0.3.2
